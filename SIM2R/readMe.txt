Pour lancer l'api qui va nourir le front, ouvrir un autre node, aller dans le dossier api  qui se trouve dans expressApp et exécuter: node app.js(attention != App.js qui est l'application qui permet de faire la simulation)
Puis aller sur le site http://localhost:3000/

Pour lancer le front end: se rendre avec node dans le dossier AppReact qui se trouve dans le dossier application et utiliser la commande "npm run start"
ensuite se diriger vers http://localhost:8080/ pour voir la page

Concernant la base de donnée, se rendre dans mysql workbench et configurer la base de donnée 
(server: 'localhost',user: 'root' password: '020295', database: 'interdependance', port: 1000, ce que j'avais mis pour ma base)
Si vous ne mettez pas la même chose, il faudra aller le changer dans le code
